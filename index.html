How does JavaScript work ?

JavaScript was initially made as a browser language only but at present it can be used to many other environment as well. Today, JavaScript has an unique positon as the most widely-adopted browser language and it's also a synchronous single threaded language. It's means it can handle one task at a time or on the other hand, a piece of code at a time. It uses dynamic typing, where type will be safety is verified at a runtime. Everything in JavaScript happens inside an execution context.

Difference between local storage vs session storage.

Local Storage: It's a property that allows JavaScript and apps to save key ans value pairs in a web browser with no expiration date.

Session Storage: It object stores data only for a session. A page session lasts as long as the web browser is open and survives over the page refresh.

Differences are given bellow:

--> Session storage is destroyed or remove once user close the browser.

--> Local storage is not destroyed or remove once user close the browser.
